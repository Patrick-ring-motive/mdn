Optional chaining assignment has a stage 1 proposal and is even implemented in the Babeljs transpiler. The next best thing is using the nullish coalescing operator to simulate the behavior. 

        
const link = document.querySelector('a');
(link??{}).href = 'https://typescripts.org';
(link?.style??{}).color = 'green';

This works by creating a dummy stand-in object when the desired object is undefined. It isn't very menory efficient if the nullish case is the more common one but that is the tradeoff for more concise and readable code.
