Shift To Boolean

A useful trick to use on any method that returns -1 for index not found is to use the Bitwise NOT ~ to adapt to a boolean operation. The following two statements are effectively equivalent. 

        
const arr = [];

if(arr.some(x => x > 5)){
  //...
}

if(~arr.findIndex(x => x > 5)){
  //...
}

This works because all numbers are considered truthy values with the exception of 0 and ~-1 == 0. Though most environments support [].findIndex, this can be useful for older environments.
